## 技术栈
**Vue3**：采用最新Vue3的语法

**Vuex**：状态管理，实现不同组件之间的状态共享

**vue-router**：路由管理，实现路由的跳转

**mock**: 用的假数据mock devServer

**axios**：发起http请求

**cube-ui**：基于 Vue.js 实现的精致移动端组件库 rem解决移动端适配问题

**Webpack**：自动化构建工具，大部分配置vue-cli脚手架已经弄好了，很方便

## 安装使用
```
npm install
```
npm run serve
```
npm run build
```
